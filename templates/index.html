About This Project

Welcome to FSCope! This project is dedicated to facilitating the analysis of homeless services data, with a focus on the following three goals:

  • Identifying Data Quality Issues
  • Visualizing Program Performance
  • Enabling User-Friendly Querying/Reporting
My original project, was written in Node.js and stored data in JSON files. This "enhanced" implementation uses Python and stores data in an SQLite database. The data used for this project is fictional and is imported from CSV-formatted spreadsheets, whose contents are shown below.

The search page allows you to filter information from this database, and select columns to view. In the Python backend, this form information is converted into a SQL query, which is executed using SQLAlchemy, and the results are formatted into Jinja2 templates using Flask. Click here to view the source code on GitHub.
