About FSCpe

Welcome to FSCope! This project is dedicated to facilitating the analysis of homeless services data, with a focus on the following three goals:

  • Identifying Data Quality Issues
  • Visualizing Program Performance
  • Enabling User-Friendly Querying/Reporting

The search page allows you to filter information from this database, and select columns to view. In the Python backend, this form information is converted into a SQL query, which is executed using SQLAlchemy, and the results are formatted into Jinja2 templates, rendered using Flask. Click here to view the source code on GitHub.
